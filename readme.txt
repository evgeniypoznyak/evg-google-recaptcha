Extended WordPress\Woocomerce Login With Google reCaptcha and hiding user/password errors


== Description ==

This plugin is implements Google reCaptcha to WordPress\Woocommerce Login page and hide user/password errors (optional).

== Installation ==


1. Install and activate plugin.


2. Go to Settings > Login with reCaptcha


3. Go to <a href="https://www.google.com/recaptcha/admin">https://www.google.com/recaptcha/admin</a>
   And get your API Keys.

4. Place your Google reCaptcha Site Key


5. Place your Google reCaptcha Secret Key


6. Choice Theme (optional)


7. Choice Size (optional)


8. Choice Hide Errors (optional)

   <ul> <li> If you choose errors "Hide", if the usernames matches - it will show that the user does not exist.
    That way, hackers won't know if your username exists. </li></ul>
